Comment crypter vos e-mails ?

salut à tous !

pour commencer qu'est-ce que c'est le pgp et à quoi sa sert ?

PGP utilise les possibilités de calcul d'un simple ordinateur personnel. Lorsqu'il est correctement utilisé, la lecture des messages 
que ce logiciel a chiffrés est impossible sans posséder la clef de déchiffrement.

PGP est gratuit, facile d'utilisation, et disponible en français.

Pour commencer

Pour commencer je vous conseille d'utiliser GPG4USB qui est trés simple d'utilisation et portable fonctionne trés bien sous 
les distribution linux et windows aussi. Donc pour le télécharger vous allez ici : GPG

Une fois téléchargé extraire le fichier .zip et ensuite exécuté le fichier linux ou windows selon votre OS.

Exemple

Ensuite une fenêtre de va s'ouvrir Premier démarrage de l'assistant la vous choisissez la langue Français 
et vous faites suivant, suivant puis terminer.

Maintenant il est temps de créer votre clefs, puis cliquer sur Gérer les clefs, clef et Générer les clefs.

ensuite une fenêtre s'ouvre et vous remplissez les champs:

- votre pseudo
- votre adresse e-mail
-  laissez vide
- laissez vide
- 4096
- votre mot de passe
- confirmation de mot de passe

Cliquer OK et patientez votre clef est entrain de se générer .

Maintenant pour récupérer votre clef publique il suffit de cocher la case à gauche de votre pseudo 
puis cliquez sur Exporter vers le presse-papiers ou vers Fichier

C'est cette clef que vous devez partage à vos interlocuteur pour qu'ils puissent vous envoyer 
des messages que seulement vous pourras déchiffrer.

Vous pouvez maintenant fermer la gestion des clefs, si vous voulais ajouter la clef public de quelqu'un 
pour pouvoir l'envoyer des messages chiffrer, cliquez sur Importer clef puis vous l'importer de la façon que 
souhaité pour ma part je fais presse-papier je trouve que c'est plus simple.

Pour écrire un message chiffré à votre interlocuteur, il suffit d'écrire votre message dans l'espace réservé 
et à droite vous cochez le pseudo de votre interlocuteur et vous cliquez par la suite sur chiffrer puis vous c
opiez la totalité du message crypté pour l'envoyé à votre destinataire.

Conclusion

Pour déchiffré un message crypter qui est pour vous, copier/collez le message dans la zone de texte et cocher la case 
à gauche du pseudo destinataire une fois celà fait cliquez sur déchiffrer et votre mot de passe vous est demandé pour pouvoir 
lire le message chiffré.

Pour finir j'espère que ce petit tuto vous aura plus !!! sur ce à Bientôt :-)
